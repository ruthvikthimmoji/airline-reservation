Install XAMPP (or any other local server stack)
Step 1: Set Up XAMPP and Start Services
Start Apache and MySQL:
Open XAMPP Control Panel.
Start the Apache and MySQL services.
Step 2: Import the Database
Import setup.sql:
Open your web browser and go to http://localhost/phpmyadmin.
Create a new database named airline_reservation.
Select the airline_reservation database.
Click on the Import tab.
Choose the setup.sql file from the db directory.
Click Go to import the database structure and data.
